# メタデータ管理ルール

## 目的

このルールは、ドキュメントのfrontmatterメタデータの管理方針を定義し、特に `last_reviewed` と `last_modified` の違いを明確化します。

---

## メタデータの種類と責務

### last_reviewed（最終レビュー日）

**意味:**
- 人間がドキュメント内容の妥当性を確認した日付

**更新主体:**
- 人間のみ

**更新タイミング:**
- ドキュメント内容をレビューし、妥当性を確認した時
- 仕様に影響する変更を行い、レビューが完了した時
- 期限切れ（90日超過）ドキュメントを再確認した時

**禁止事項:**
- ❌ 自動更新（スクリプト、CI、Hooks）
- ❌ AIによる自動更新
- ❌ 内容確認なしの日付更新
- ❌ 「とりあえず」の日付更新

**理由:**
- `last_reviewed` は「内容が妥当であることを人間が保証した日」を意味する
- 自動更新すると、この保証が失われる
- 編集日 ≠ レビュー日

### last_modified（最終更新日）

**意味:**
- ドキュメントファイルが最後に編集された日時

**更新主体:**
- 自動（Git、エディタ、スクリプト）

**更新タイミング:**
- ファイルが保存されるたび

**注意:**
- このプロジェクトでは `last_modified` は必須ではない
- 必要に応じてGitのコミット履歴から取得可能

### owner（責任者）

**意味:**
- ドキュメントの更新・レビューに責任を持つ個人またはチーム

**設定値:**
- 実在の個人名（例: `taro-yamada`）
- 実在のチーム名（例: `platform-team`, `api-team`）

**禁止値:**
- ❌ `unknown`
- ❌ `tbd` / `TBD`
- ❌ 空文字
- ❌ プレースホルダー

**責務:**
- ドキュメント内容の妥当性維持
- 期限切れ時の再レビュー
- 関連する実装変更時の更新判断

**変更タイミング:**
- 責任者の引き継ぎ時のみ
- 組織変更時のみ

**注意:**
- `owner` は「作成者」ではない
- `owner` は「現在の責任者」を示す

### title（タイトル）

**意味:**
- ドキュメントの表題

**設定値:**
- 人間が読んで内容が分かるタイトル
- 簡潔で明確な表現

**注意:**
- 重複titleは推奨されない（統合品質チェックで検出）

### status（ステータス）

**意味:**
- ドキュメントの状態

**設定値:**
- `approved`: 正式仕様
- `draft`: 草案
- その他: プロジェクト固有の値

**変更タイミング:**
- ドキュメントのライフサイクルに応じて変更
- 詳細は document-lifecycle.md 参照

---

## メタデータ更新の判断フロー

### ドキュメントを編集した場合

```
編集内容は仕様に影響するか？
├─ YES → 人間レビュー実施 → last_reviewed 更新
└─ NO（タイポ修正、表現調整のみ）→ last_reviewed 更新不要
```

### 期限切れ（90日超過）の場合

```
1. ドキュメント内容を確認
2. 現在も妥当か判断
   ├─ 妥当 → last_reviewed を本日に更新
   └─ 不適切 → 内容を修正 → レビュー → last_reviewed 更新
```

## AIエージェントへの指示

AIエージェント（Kiro等）がドキュメントを編集する場合：

**MUST（必須）:**
- `status: approved` のドキュメントを編集する場合、必須メタデータを保持する
- メタデータの順序を変更しない
- メタデータのキーを削除しない

**MUST NOT（禁止）:**
- `last_reviewed` を自動更新してはならない
- `owner` を勝手に変更してはならない
- `status` を指示なく変更してはならない

**SHOULD（推奨）:**
- 仕様に影響する変更を行った場合、人間にレビューを促す
- 期限切れドキュメントを発見した場合、ownerに通知する

---

## 例外

以下の場合は、このルールの適用外とすることができます：

- プロジェクト固有の事情により、別のメタデータ体系を採用する場合
- 自動生成ドキュメント（OpenAPI等）で、別の管理方式を採用する場合

例外を設ける場合は、プロジェクトのREADMEまたは別のルール文書に明記してください。
